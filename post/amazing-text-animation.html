Amazing Text Animation using CSS only

  • August 15, 2020
  • HTML and CSS design

Soure Code

HTML

<!DOCTYPE html>
  <html lang="en" dir="ltr">
    <head>
      <meta charset="utf-8">
      <title>Amazing text animation</title>
      <meta name="viewport" content="width=device-width,initial-scale=1">
      <link rel="stylesheet" href="style.css">
    </head>
    <body>
      <div class="ani-text">
        <p>
          <span>
            Catalyst Coder
          </span>
        </p>
      </div>
    </body>
  </html>

CSS


body{
  background: #151414;
}
.ani-text{
  position: absolute;
  top:50%;
  left:50%;
  transform: translate(-50%,-50%);
}
p{
  border: 4px double rgb(255,255,255,0.25);
  border-width: 4px 0;
  padding: 25px 0px;
}
p span{
  font: 700 4em/1 "Oswals", sans-serif;
  letter-spacing: 0;
  display: block;
  margin: 0 auto;
  text-shadow: 0 0 80px rgba(255,255,255,0.5);
  background: url(animated_text_fill.png);
  -webkit-background-clip: text;
  -webkit-text-fill-color: transparent;
  -webkit-animation: slide 80s linear infinite;
  -webkit-transform: translate3d(0,0,0);
  -webkit-backface-visibility: hidden;
}
@-webkit-keyframes slide{
  0%{
    background-position: 0% 50%;
  }
  100%{
    background-position: 100% 50%;
  }
}

Note this awsome design doesn’t require any Javascript files. So you can easily copy and paste the code in your website.
